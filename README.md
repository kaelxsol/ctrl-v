## Features

- **One-Click Clone** — Ctrl+Click any token on Axiom, metadata fills instantly
- **Multi-Platform** — Launch on Pump.fun, Bonk.fun, or Bags.fm
- **Wallet Support** — Phantom, Solflare, private key import, or burner wallets
- **Jito Bundles** — Atomic multi-wallet launches in a single block
- **AI Opposites** — Alt+Click generates an AI-powered opposite token concept
- **Open Source** — No fees, no token, no tracking

## Install

1. **[Download Latest Release](../../releases/latest)** — grab the `.zip`
2. Go to `chrome://extensions` → enable Developer Mode
3. Click **Load unpacked** → select the extracted folder
4. Visit [axiom.trade](https://axiom.trade) → Ctrl+Click any token

### From Source

```bash
git clone https://github.com/kaelxsol/ctrl-v.git
```

Load unpacked from the cloned folder.

## Usage

1. Browse any token on axiom.trade
2. **Ctrl+Click** to clone, **Alt+Click** for AI opposite
3. Edit metadata if needed in the CTRL-V panel
4. Pick platform → connect wallet → launch

## Security

- Never share your private keys
- Burner wallets are stored locally — export before clearing data
- Review all transactions before signing
- Use at your own risk — launches involve real funds
